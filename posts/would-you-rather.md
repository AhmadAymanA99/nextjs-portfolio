## Introduction:
Would You Rather is an interactive polling web app built to strengthen expertise in state management with Redux. The application allows users to create polls, vote on questions, and see results in real-time.

### Motivation:
This project was developed to gain hands-on experience with Redux for state management in React applications. It demonstrates understanding of complex state management patterns, middleware, and asynchronous actions.

### Features:
• Create and answer "Would You Rather" questions
• View leaderboard based on user activity
• Real-time poll results and statistics
• User authentication and profile management

### Technologies:
React, Redux

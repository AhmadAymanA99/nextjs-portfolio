## Introduction:
Covid-19 Dashboard is a dashboard tracking live COVID-19 cases worldwide. The application was initially developed in React Native and later optimized for web using React.

### Motivation:
• Our role as developers to contribute during the pandemic
• Awareness is more important than cure
• Reduced pressure on hospitals by providing information

### Why this Project:
The only way to fight this pandemic is by avoiding large gatherings and by not going to high infected rate areas. The application provides us with the latest numbers of the infection per country and provides the country location on the map.

### Technologies:
React, Firebase, API Integration
